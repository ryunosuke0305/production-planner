# 一般利用者向け

## 目的別の操作ガイド

### 生産計画を確認する
- 左上のメニューから「スケジュール」を選択します。
- 週表示（1時間 / 2時間 / 日単位）は画面上部のボタンで切り替えます。
- 右側の「日次在庫」欄で、日別の在庫推移を確認できます。

### ブロックを追加・編集する
- ガントチャート上で空いている時間帯をクリックし、「計画編集」モーダルを開きます。
- 品目・数量・メモを入力し、保存してください。
- 既存ブロックはドラッグで移動、左右端をドラッグすると長さを変更できます。

### 品目情報を確認する
- 「マスタ管理」画面を開き、「品目一覧」を選択して品目マスタの内容を確認します。
- 品目のレシピ（使用原料）や在庫関連の設定値もここで参照できます。
- 品目コード（品目ID）は保存され、再読み込み後も保持されます。

### 日別在庫や受注を取り込む
- 「Excel取り込み」画面で日別在庫・受注一覧のファイルを読み込みます。
- 読み込むファイルは `.xlsx` / `.xls` / `.csv` に対応しています。

### チャットで計画を更新する（任意）
- スケジュール画面右側のチャット欄に指示を入力し送信します。
- 例: `品目コード A を 9/12 10:00から2時間、40cs 追加して`
- 実行結果はチャットに返され、必要に応じてブロックが追加・更新されます。

### JSON をエクスポートする
- スケジュール画面の「JSONエクスポート」ボタンを押します。
- 生成AI連携や外部システム連携向けのJSONがダウンロードされます。

## 操作時の注意点
- 変更内容は自動的に保存されますが、通信エラーがある場合は画面上の通知を確認してください。
- 画面上部の週移動ボタンで週を切り替えた後は、必ず内容を確認してから編集してください。
